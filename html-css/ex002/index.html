Parágrafos e quebras de linha

Você pode escrever
um parágrafo de
qualquer jeito.
Basta colocar tudo
no meio do par de tags
<p> e </p>.

Para deixar o texto como linhas separadas igual ta acima, antes de cada linha voce deve por a tag <br> (que se chama quebra de linha).

Adicionando alguns simbolos especiais para exemplo:

© - SimboloDeCopyright
